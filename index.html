Orden del Día

El orden original corresponde al enviado en la citación. El orden de sesión conserva los cambios realizados al inicio o durante la reunión y es el que se usa para votaciones y acta.

Orden del día original / citación

Base histórica usada para la citación. No se modifica desde sesión.

    Orden del día de sesión / votación

    Versión vigente para la reunión, votaciones y acta. Los cambios quedan trazados. Los usuarios autorizados pueden arrastrar los puntos para cambiar su posición.

        Alertas de citaciones pendientes

        Configura la hora diaria en la que el sistema debe recordar a los administradores y al Secretario del Consejo las sesiones que siguen pendientes por citar. La programación se guarda en la hoja Configuracion y reprograma el trigger automático.

        Ejemplo: 10:30 a. m. hora Colombia.

        Nota

        Apps Script no garantiza ejecución exacta al minuto. El trigger diario puede ejecutarse algunos minutos alrededor de la hora configurada. Para validar la ejecución automática sin esperar al siguiente día, use el botón de prueba de 2 minutos.

        Integridad del sistema de votaciones

        Consolida votaciones duplicadas heredadas, relaciona el punto original con el punto de sesión, tipifica los órganos y anula votos de personas que no tengan derecho vigente. En comités solo conserva como votantes a los Consejeros Integrantes del Comité.

        Carga de Datos de Inicialización

        Esta acción es irreversible y ejecutará el script de poblamiento inicial que cargará el cronograma de 2026, los comités estándar, plantillas base y sus respectivos miembros por defecto. Solo debe ejecutarse si la base de datos está totalmente limpia.

        Ficha de Persona

        Si selecciona "Otro", se habilitará un campo para escribir el cargo exacto.

        Para firmar citaciones se usará la imagen registrada en la persona activa con cargo Presidente. Variable en plantilla: {{FIRMA}}.

        Programar / Editar Sesión

        Este campo solo aplica para sesiones del Consejo Directivo, ordinarias o extraordinarias. Variable para la plantilla: {{NUMERO_SESION}}.

        Se completa con la persona activa del Directorio General cuyo cargo institucional sea Presidente.
